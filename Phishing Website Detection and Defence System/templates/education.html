What is Phishing?

Phishing is a type of cyber attack that uses disguised emails, messages, or websites to trick recipients into revealing sensitive information. Attackers often pose as trustworthy entities to steal data such as login credentials, credit card numbers, or personal information.

Common Tactics

Phishers use tactics like creating urgency, impersonating trusted organizations, using misleading domain names, and exploiting current events. They often include deceptive links that lead to fake websites designed to steal your information.

How to Stay Safe

Protect yourself by verifying sender information, hovering over links before clicking, being cautious with attachments, using multi-factor authentication, and keeping your software updated. When in doubt, contact organizations directly.

How to Identify Phishing URLs

Red Flags in URLs

  • Misspelled Domain Names: Slight variations like "paypai.com" instead of "paypal.com"
  • IP Addresses in URL: URLs containing IP addresses (e.g., http://123.45.67.89/...) instead of domain names
  • Suspicious Subdomains: "bank.secure-login.com" instead of "secure-login.bank.com"
  • Excessive Subdomains: Long, complex domain structures designed to confuse
  • URL Shorteners: Shortened URLs that hide the true destination

Safety Indicators

  • HTTPS Connection: Secure websites use HTTPS and show a padlock icon in the browser
  • Proper Domain Structure: Main domain name should match the organization you expect
  • Consistent URL: URL matches what you'd expect for the service you're accessing
  • Few Special Characters: Legitimate URLs typically avoid excessive symbols like @, %, or extra hyphens
  • Established Domain Age: Legitimate organizations typically have domains registered for many years

Examples of Phishing URLs

Phishing Example:

https://paypal-secure-login.com/

Why it's suspicious: This URL uses "paypal" in the domain, but it's not the real PayPal.com. The legitimate URL would be "https://www.paypal.com/".

Phishing Example:

https://accounts-google.com/login

Why it's suspicious: This looks like a Google login page but is not hosted on google.com. The legitimate URL would be "https://accounts.google.com/".

Phishing Example:

http://secure-bank-login.com@185.142.236.34/

Why it's suspicious: This URL uses the @ symbol to hide the actual destination (the IP address). The text before @ is ignored when resolving where to navigate.

Safety Tips

Verify the URL

Always check the full URL in your browser's address bar before entering sensitive information. Look closely at the domain name and be suspicious of misspellings or added words.

Don't Trust Link Text

Hover over links without clicking to see where they actually lead. The displayed text may say "PayPal.com" but the actual link could be entirely different.

Use Bookmarks

For important sites like banking, email, or shopping, create bookmarks instead of clicking links from emails or messages.
